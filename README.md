# Sample Hardhat Project

This project demonstrates a basic Hardhat use case. It comes with easy challenge contract, solve the challenge by rendering the balance to zero.

to run the challenge follow the steps below:

1- run a local node
```shell
npx hardhat node
```
2- in another cmd/shell run: 
```shell
npx hardhat ignition deploy ignition\modules\deploy.js --network localhost
```
3- run node js server 
```shell
node server.js
```
4- to see the frontend use vscode go live plugin to run contract.html 
or run python3 server on any desried port 
```shell 
python3 -m http.server 8080
```

happy hacking !!!!
